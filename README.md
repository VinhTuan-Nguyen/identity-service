# Identity Service API Application
## 1. Run App Guideline
- When you cloned this app to your machine: execute `mvn clean install -DskipTests`
- Ensure that the app build is successfully
- Run this command to start the app:`mvn spring-boot:run --define spring-boot.run.arguments="--spring.profiles.active=local"`
